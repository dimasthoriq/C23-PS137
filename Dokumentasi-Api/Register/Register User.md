# Register User

Memasukkan hasil pendaftaran user kedalam database

**URL** : `/api/register`

**Method** : POST

**Body** :
```json
{
    "name": "String",
    "email": "String",
    "password": "String"
}
```
## Success Responses
**Condition** : Data telah masuk kedalam database

**Code** : `200 OK`
